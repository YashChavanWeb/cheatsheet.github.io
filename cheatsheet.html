  5. For bringing a line under the navigation bar items with transition
    6. 
.classname ul li::after{
    content:'';
    width: 0%;
    height: 2px;
    display: block;
    margin: auto;
    transition: 0.5s;
}

.classname ul li:hover::after{
    width: 100%;
}


  6. For bringing text in the center of Web Page
    7.

  7. For Making a Button
    8. 
.button{
    background: transparent;
    cursor: pointer;
}

button:hover{
    background: red;
    transition: 0.3s;
}


  8. Java Script for a Responsive Menu
    9. 
<script>
    var idname = document.getElementById("idname")

    function showmenu(){
        idname.style.right="0"
    }

    function hidemenu(){
        idname.style.right="-200px"
    }
</script>

<!-------------- Adding tags for Applying the above Js-------------->

Before ul 
<i class="fa fa-times" oneclick="hidemenu()"></i>

After /div which is after /ul 
<i class="fa fa-bars" oneclick="showmenu()"></i>

<!-------------- Adding a link for Applying the above Js-------------->



<!-------------- Adding CSS for Applying the above Js-------------->
nav .fa{
    display: none;
}

@media(max-width: 700px){
text-size: change;

    width: 200px;
    top: 0%;
    right: -200px;
    text-align: left;
    z-index: 2;
    transition: 0.5s;
}

nav .fa{
    display:block;
    margin: 10px;
    cursor: pointer;
}

    padding: 30px;
}
    }


  9. Creating Good Boxes
    10. 
.classname{  <!--This is selecting the whole container-->
    display: flex;
    margin: 5%;
    justify-content: space-between;
}

.classname{ <!--This is selecting each box-->
    flex-basis: 31%;
    box-sizing: border-box;
    padding: 20px 12px;
    border-radius: 10px;
    transition: .3s;
}

<!--------------For making these boxes responsive---------------->

@media(max-width: 700px){
.classname{ <!--This is selecting the whole container-->
    flex-direction: column;
}
}

<!--------------Another type of boxes which are images---------------->

.classname{ <!--The class name should be only of div of that particular box in which
the image is there-->
    flex-basis: 32%;
    border-radius: 10px;
    margin-bottom: 30px;
    position: relative;
    overflow: hidden;
}

.classname img{
    width: 100%;
    display: block;
}

  11. To add a hovering layer on the image
    12.

    }
    
    .classname:hover{

    }


  12. To add a Text at the bottom center of the image/box
    13. 
    .classname h3{
        width: 100%;
        color: white;
        font-size: 25px;
        bottom: 0;
        left: 50%;
        transform: translateX(-50%);
        position: absolute;
        opacity: 0; .layer:hover h3{
        bottom: 50%;
        opacity: 0; <!--This is the main reason for this effect-->
        transition: 0.5s;
    }
    }

    <!--------------For creating a hovering effect here---------------->
    
    .classname:hover h3{ <!--For creating hover effect we have to write h3 after hover-->
        bottom: 50%;
        opacity: 1;
    }

  14. How to add stars to your website
    15. 

Once you have added the font awesome link just paste

<i class="fa fa-star"></i>   (one i = one full star)
<i class="fa fa-star-o"></i> (one i = one empty star)
<i class="fa fa-star-half-o"></i> (one i = one half star)
